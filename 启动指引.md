# Meowone 项目启动指南

## 项目概述

Meowone 是一个智能体协作平台，包含：

- **前端**：Next.js 管理后台
- **后端**：FastAPI API 服务
- **智能体**：三个 A2A 协议智能体
  - `code-writer`：代码编写智能体
  - `doc-assistant`：文档助手智能体
  - `mermaid-assistant`：Mermaid 图表生成智能体

---

## 环境要求

- Node.js 18+
- Python 3.11+
- pnpm（前端包管理器）
- uv（Python 包管理器）

---

## 启动步骤

### 1. 启动后端服务

```bash
cd backend
source .venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

后端服务地址：http://localhost:8000

### 2. 启动前端服务

```bash
cd admin_web
pnpm dev
```

前端服务地址：http://localhost:3000

### 3. 启动智能体服务

#### code-writer 智能体

```bash
cd agents/code-writer
source .venv/bin/activate
python __main__.py
```

服务地址：http://localhost:8001

#### doc-assistant 智能体

```bash
cd agents/doc-assistant
source .venv/bin/activate
python __main__.py
```

服务地址：http://localhost:8002

#### mermaid-assistant 智能体

```bash
cd agents/mermaid-assistant
source .venv/bin/activate
python __main__.py
```

服务地址：http://localhost:8003

## 快速启动脚本

如需同时启动所有服务，可以使用后台运行方式：

```bash
# 终端 1 - 后端
cd backend && source .venv/bin/activate && uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# 终端 2 - 前端
cd admin_web && pnpm dev

# 终端 3 - 智能体们
cd agents/code-writer && source .venv/bin/activate && python __main__.py
cd agents/doc-assistant && source .venv/bin/activate && python __main__.py
cd agents/mermaid-assistant && source .venv/bin/activate && python __main__.py
```
