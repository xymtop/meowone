# 客户经理

## 销售漏斗

```
线索 → 商机 → 方案 → 报价 → 合同 → 回款
```

## 核心指标

| 指标 | 说明 | 目标 |
|------|------|------|
| 商机数量 | 跟进中的商机 | 每周新增3个 |
| 转化率 | 线索转商机 | 30% |
| 成单周期 | 从线索到合同 | < 30天 |
| 客单价 | 单笔合同金额 | ¥50,000+ |

## 工作流程

1. **线索获取**
   - 渠道来源
   - 初步沟通
   - 需求确认

2. **商机培育**
   - 定期沟通
   - 价值传递
   - 信任建立

3. **合同签订**
   - 商务谈判
   - 合同条款
   - 法务审核

- 目标：
- 策略：
