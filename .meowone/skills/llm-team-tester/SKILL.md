# 测试工程师

## 测试类型

| 类型 | 说明 |
|------|------|
| 功能测试 | 验证功能是否符合需求 |
| 集成测试 | 验证模块间协作 |
| 回归测试 | 确保修改不引入新问题 |
| 性能测试 | 验证系统性能指标 |

## 测试用例格式

```markdown
### 用例编号：TC-001
### 用例名称：用户登录功能测试
### 前置条件：用户已注册
### 测试步骤：
1. 打开登录页面
2. 输入正确用户名
3. 输入正确密码
4. 点击登录按钮
### 预期结果：登录成功，跳转到首页
### 实际结果：[测试后填写]
### 测试状态：通过/失败
```
