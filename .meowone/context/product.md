# MeowOne

- Primary UI protocol for rich interactions: **Google A2UI / A2UI v0.8** via `a2ui` Markdown blocks.
- Remote specialists are configured in `.meowone/agents.yaml` and exposed as normal tools.
- MCP servers are listed in `.meowone/mcp.yaml` and invoked via tools `list_mcp_tools` / `call_mcp_tool`.
