# MeowOne 版本升级 v3 — 智能体编排与调度系统

## 1. 升级背景

### 1.1 当前系统状态

MeowOne v2 已实现：
- **智能体工厂化**：通过 Prompt + MCP + Skill + Model + Tools + Loop 组合快速构建智能体
- **基础调度能力**：Direct、MasterSlave、Swarm 三种调度模式
- **平台化管理**：后端 API + 前端管理界面

### 1.2 v3 升级目标

将 MeowOne 从 **"多智能体平台"** 升级为 **"智能体操作系统"**：

> **智能体 = 一等公民（可独立存在、可组合、可调度）**
> 
> **操作系统 = 智能体₁ + 智能体₂ + ... + 调度策略 + 环境**

核心升级方向：
1. **组织化管理**：像管理公司一样管理智能体（组织 → 团队 → 成员）
2. **插件化调度策略**：用户可新增、配置自定义调度算法
3. **插件化 Loop**：用户可切换不同的推理/思考模式
4. **环境隔离**：为不同任务提供安全的执行环境

## 2. 核心概念设计

### 2.1 概念类比

| 现实世界 | MeowOne v3 | 说明 |
|---------|------------|------|
| 公司/组织 | **Organization** | 顶层组织单元 |
| 部门/团队 | **Team** | 职能团队，包含多个智能体 |
| 员工 | **Agent** | 一等公民，可属于多个团队 |
| 项目经理 | **Team Leader** | 团队协调者，负责任务分配 |
| 任务分配 | **Dispatcher** | 调度器，选择执行策略 |
| 调度算法 | **Strategy（插件）** | 可插拔的调度算法 |
| 工作流程 | **Loop（插件）** | 可插拔的推理/执行模式 |
| 办公环境 | **Environment** | 执行环境（沙箱、权限、资源） |

### 2.2 核心公式

```
智能体 = 提示词 + MCP + Skill + Model + 工具 + Loop（均插件化）

操作系统 = 智能体₁ + 智能体₂ + ... + 调度策略 + 环境
                         ↓                ↓
                   多个智能体集合    可插拔组件
```

## 4. 插件系统设计

### 4.1 Loop 插件接口

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass
from typing import Any, Dict, List, AsyncIterator

@dataclass
class LoopResult:
    output: str
    stop_reason: str
    loop_rounds: int
    duration_ms: int

@dataclass
class StepResult:
    step_number: int
    output: Any
    is_final: bool

class BaseLoop(ABC):
    """Loop 插件基类"""
    
    name: str
    description: str
    config_schema: Dict[str, Any]
    
    @abstractmethod
    async def run(
        self,
        agent_id: str,
        user_input: str,
        history: List[Dict],
        context: Dict[str, Any]
    ) -> LoopResult:
        """执行完整 Loop"""
        pass
    
    @abstractmethod
    async def step(
        self,
        state: Dict[str, Any],
        context: Dict[str, Any]
    ) -> StepResult:
        """单步执行"""
        pass
    
    async def on_tool_call(self, tool_name: str, params: Dict, context: Dict) -> Any:
        """工具调用钩子"""
        pass
    
    async def on_llm_call(self, messages: List[Dict], context: Dict) -> Any:
        """LLM 调用钩子"""
        pass
    
    async def should_continue(self, state: Dict, context: Dict) -> bool:
        """判断是否继续执行"""
        pass
```

### 4.2 Strategy 调度策略插件接口

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass
from typing import List, Dict, Any, Optional

@dataclass
class SelectionResult:
    target_id: str                    # 选中的目标（Agent/Team）
    target_type: str                  # agent / team / org
    reasoning: str                    # 选择理由
    confidence: float                 # 置信度 0-1

@dataclass
class DispatchResult:
    success: bool
    execution_id: str
    output: Any
    error: Optional[str]

class BaseStrategy(ABC):
    """调度策略插件基类"""
    
    name: str
    description: str
    config_schema: Dict[str, Any]
    
    @abstractmethod
    async def select(
        self,
        task: str,
        task_requirements: Dict[str, Any],
        targets: List[Dict[str, Any]],
        context: Dict[str, Any]
    ) -> SelectionResult:
        """选择最佳执行目标"""
        pass
    
    @abstractmethod
    async def dispatch(
        self,
        task: str,
        target: Dict[str, Any],
        context: Dict[str, Any]
    ) -> DispatchResult:
        """分发任务到目标"""
        pass
    
    async def on_complete(self, result: Any, context: Dict) -> Any:
        """完成回调"""
        pass
    
    async def on_error(self, error: Exception, target: Dict, context: Dict) -> Any:
        """错误处理"""
        pass
```

### 4.3 插件注册与加载

```python
class PluginRegistry:
    """插件注册表"""
    
    def __init__(self):
        self._loops: Dict[str, type[BaseLoop]] = {}
        self._strategies: Dict[str, type[BaseStrategy]] = {}
    
    def register_loop(self, loop_class: type[BaseLoop]) -> None:
        """注册 Loop 插件"""
        self._loops[loop_class.name] = loop_class
    
    def register_strategy(self, strategy_class: type[BaseStrategy]) -> None:
        """注册调度策略插件"""
        self._strategies[strategy_class.name] = strategy_class
    
    def get_loop(self, name: str) -> Optional[type[BaseLoop]]:
        return self._loops.get(name)
    
    def get_strategy(self, name: str) -> Optional[type[BaseStrategy]]:
        return self._strategies.get(name)
    
    def list_loops(self) -> List[str]:
        return list(self._loops.keys())
    
    def list_strategies(self) -> List[str]:
        return list(self._strategies.keys())

# 全局注册表
plugin_registry = PluginRegistry()

# 自动发现并注册内置插件
def discover_plugins():
    # 注册内置 Loop
    from app.loops import react, plan_exec, multi_agent_debate
    plugin_registry.register_loop(react.ReActLoop)
    plugin_registry.register_loop(plan_exec.PlanExecLoop)
    plugin_registry.register_loop(multi_agent_debate.MultiAgentDebateLoop)
    
    # 注册内置策略
    from app.scheduler.strategies import direct, round_robin, auction
    plugin_registry.register_strategy(direct.DirectStrategy)
    plugin_registry.register_strategy(round_robin.RoundRobinStrategy)
    plugin_registry.register_strategy(auction.AuctionStrategy)
```

## 5. 操作系统核心设计

### 5.1 Orchestrator（操作系统核心）

```python
class Orchestrator:
    """智能体操作系统核心"""
    
    def __init__(
        self,
        agents: List[Agent],
        organizations: List[Organization],
        teams: List[Team],
        strategy: BaseStrategy,
        environment: Environment,
        loop: BaseLoop,
    ):
        self.agents = {a.id: a for a in agents}
        self.organizations = {o.id: o for o in organizations}
        self.teams = {t.id: t for t in teams}
        self.strategy = strategy
        self.environment = environment
        self.loop = loop
        self.plugin_registry = plugin_registry
    
    async def run(self, task: str, target: Optional[Dict] = None) -> OrchestratorResult:
        """执行任务"""
        # 1. 解析任务
        parsed = await self._parse_task(task)
        
        # 2. 获取候选目标
        candidates = self._get_candidates(target, parsed)
        
        # 3. 选择执行策略
        if isinstance(self.strategy, TeamDispatchStrategy):
            selected = await self._team_dispatch(parsed, candidates)
        elif isinstance(self.strategy, CapabilityMatchStrategy):
            selected = await self._capability_match(parsed, candidates)
        else:
            selected = candidates[0] if candidates else None
        
        # 4. 执行任务
        result = await self._execute(selected, parsed)
        
        return result
    
    async def _team_dispatch(self, task, candidates) -> List[Dict]:
        """团队分发模式"""
        # 识别需要团队协作
        team = self._select_team(task, candidates)
        
        # 任务分解
        subtasks = await self._decompose_task(task)
        
        # 并行/串行执行
        results = []
        for subtask in subtasks:
            member = await self.strategy.select(subtask, team.members, {})
            result = await self._execute(member, subtask)
            results.append(result)
        
        # 结果聚合
        return await self._aggregate_results(results)
    
    async def _capability_match(self, task, candidates) -> Dict:
        """能力匹配模式"""
        requirements = self._extract_requirements(task)
        return await self.strategy.select(task, requirements, candidates, {})
```

### 5.2 任务执行流程

```
用户输入任务
     ↓
任务解析（意图识别、需求提取）
     ↓
候选目标收集（Agent/Team/Organization）
     ↓
调度策略选择
┌────────────────────────────────────────────────────┐
│  策略1: Direct → 直接执行                           │
│  策略2: Team → 团队分发 → 任务分解 → 成员执行 → 汇总  │
│  策略3: Capability → 能力匹配 → 最优选择 → 执行      │
│  策略4: Auction → 竞拍 → 智能体投标 → 最高权 → 执行  │
│  策略5: Democratic → 多方协商 → 共识 → 执行           │
└────────────────────────────────────────────────────┘
     ↓
Loop 执行（推理 + 工具调用）
     ↓
结果聚合
     ↓
返回结果
```

MeowOne v3 将从 **"智能体平台"** 升级为 **"智能体操作系统"**，核心价值：
