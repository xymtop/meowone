# MeowOne 后端接口文档（更新版）

本文档基于当前 `backend/app/api` 与最新 Agent 改造实现整理，去除历史重复段落，供前后端联调使用。

## 1. 基本信息

- Base URL: `http://<host>:<port>`
- 协议: HTTP/JSON，流式接口为 SSE（`text/event-stream`）
- CORS: 全开放
- 鉴权: 当前未启用
- 通用错误格式:

```json
{
  "detail": "error message"
}
```

## 2. 健康检查

### GET `/health`

```json
{
  "status": "ok"
}
```

## 3. 会话与消息

### POST `/api/sessions`

创建会话。

### GET `/api/sessions`

会话列表。

### GET `/api/sessions/{session_id}`

会话详情（不存在返回 `404`）。

### PATCH `/api/sessions/{session_id}`

更新会话标题（不存在返回 `404`）。

### DELETE `/api/sessions/{session_id}`

删除会话与关联数据。

### GET `/api/sessions/{session_id}/messages`

分页查询消息。

Query:

- `limit` 默认 `50`
- `offset` 默认 `0`

## 4. 对话与网关（SSE）

### POST `/api/sessions/{session_id}/chat`

发起会话轮次。

### POST `/api/sessions/{session_id}/card-action`

卡片动作回传后继续执行。

### POST `/api/sessions/{session_id}/a2ui-action`

A2UI 动作回传后继续执行。

### POST `/api/gateway/turn`

网关统一入口。`session_id` 与 `external_thread_id` 至少一个。

### GET `/api/gateway/logs`

网关日志轮询。

Query:

- `session_id` 可选
- `cursor` 默认 `0`
- `limit` 默认 `50`（最大 `200`）

SSE 事件：

- `thinking`
- `delta`
- `card`
- `tool_call`
- `tool_result`
- `error`
- `done`

## 5. OpenAI 兼容

### POST `/v1/chat/completions`

- 兼容 OpenAI Chat Completions
- `stream=true` 返回流式 chunk
- 支持扩展参数：`scheduler_mode`、`task_tag`、`max_rounds`、`max_tool_phases`、`timeout_seconds`

## 11. Agent 配置管理（`/api/agents`）

### GET `/api/agents?agent_type=internal|external`

查询 Agent 列表。

### POST `/api/agents/internal`

新增/更新内部 Agent 配置。

请求体（新增 `prompt_key`）：

```json
{
  "name": "writer",
  "description": "写作助手",
  "system_prompt": "你是写作助手",
  "prompt_key": "agent.writer.base",
  "mcp_servers": ["user-apple-mcp"],
  "agent_skills": ["analysis-skill"],
  "allow_tools": ["call_mcp_tool"],
  "deny_tools": ["bash"],
  "max_rounds": 8,
  "max_tool_phases": 8,
  "timeout_seconds": 120
}
```

### POST `/api/agents/external`

新增/更新外部 Agent 配置。

请求体（新增 `protocol`，当前仅支持 `a2a`）：

```json
{
  "name": "researcher_remote",
  "description": "外部研究助手",
  "base_url": "https://example-agent.com",
  "protocol": "a2a"
}
```

错误：

- `400`: `only a2a protocol is supported for external agents`

### DELETE `/api/agents/{agent_type}/{name}`

删除 Agent。

## 12. Internal Agent 运行接口（`/api/internal-agents`）

### GET `/api/internal-agents`

查询运行时内部 Agent 列表。

### POST `/api/internal-agents`

创建内部 Agent（同样支持 `prompt_key` 字段）。

### POST `/api/internal-agents/{agent_name}/invoke`

调用内部 Agent。

请求体：

```json
{
  "task": "请总结今天的日报",
  "history": []
}
```

## 13. 统一 Agent 执行接口（新增）

前缀：`/api/agent-executions`

### POST `/api/agent-executions`

统一执行 internal/external Agent。

请求体：

```json
{
  "agent_name": "writer",
  "task": "请输出一份日报大纲",
  "history": [],
  "overrides": {
    "max_rounds": 6,
    "max_tool_phases": 6,
    "timeout_seconds": 90
  }
}
```

成功响应（统一结构）：

```json
{
  "ok": true,
  "agent_name": "writer",
  "agent_type": "internal",
  "output": "......",
  "duration_ms": 1234,
  "loop_rounds": 3,
  "error": null,
  "error_code": null,
  "execution_id": "uuid"
}
```

外部 Agent 失败时的错误码（`detail.error_code`）：

- `A2A_UNREACHABLE`
- `A2A_TIMEOUT`
- `A2A_PROTOCOL_ERROR`
