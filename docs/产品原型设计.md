# 🔄 七、关键交互动线（Figma需要连线）

---

## 1️⃣ 新建会话

```
点击 [+ New Chat]
→ 清空 Chat Area
→ 创建新会话
```

---

## 2️⃣ 发送消息

```
输入 → 点击发送
→ 出现用户气泡
→ 系统loading
→ 返回消息/卡片
```

---

## 3️⃣ 点击卡片按钮（核心）

```
点击 [确认]

→ 卡片进入 loading 状态
→ 生成系统消息：
   “用户确认创建会议”
→ 返回结果
```

---

## 4️⃣ 表单提交

```
填写 → 提交

→ 转换为用户消息
→ 继续对话流程
```

# 🎨 八、Figma组件建议（非常重要）

---

## 建议组件化结构

```
Components/
├── Chat/
│   ├── UserBubble
│   ├── AssistantBubble
│
├── Card/
│   ├── InfoCard
│   ├── ActionCard
│   ├── FormCard
│
├── Layout/
│   ├── Sidebar
│   ├── TopBar
│   ├── InputBar
```

---

## 设计原则

* 全部用 **Auto Layout**
* 卡片宽度固定（PC：480px）
* 间距统一（8 / 16 / 24）

---

# 🧠 最关键一句总结

这个原型的本质是：

> **把“聊天界面”做成“操作系统桌面”，
> 把“卡片”做成“应用界面”。**
